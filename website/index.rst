Systèmes informatiques 
======================
 
.. only:: html

	Ce site web contient les supports des cours de systèmes informatique donné aux `étudiants en informatique <http://www.uclouvain.be/info.html>`_ à l'`Université catholique de Louvain <http://www.uclouvain.be>`_ (UCL):
	
	<ul>
	  <li>Le <a href="https://uclouvain.be/cours-2020-lepl1503">projet 3 (LEPL1503)</a> donné en deuxième année, second quadrimestre, des cycles ingénieurs option informatique et sciences informatiques;</li>
	  <li>Le cours <a href="https://uclouvain.be/cours-2020-LINFO1252">Systèmes informatique (LINGO1252)</a> donné lui en troisième année, premier quadrimestre.</li>
	</ul>
	
	 Les étudiants sont invités à ajouter leur commentaires en soumettant des patches via https://github.com/obonaventure/SystemesInformatiques.
	
Vous pouvez accéder aux différentes parties du syllabus via les liens dans la colonne de gauche.
